# GoF Design Patterns in TypeScript

Даний проєкт демонструє реалізацію 6 класичних патернів проєктування (Gang of Four).

## Реалізовані патерни
1. **Creational**: Singleton, Factory Method.
2. **Structural**: Adapter, Facade.
3. **Behavioral**: Strategy, Observer.

## Як запустити
1. Встановіть залежності: `npm install`
2. Запустіть через ts-node: `npx ts-node index.ts`

## Висновки
Патерни проєктування дозволяють зробити код гнучким до змін. 
- **Singleton** економить ресурси.
- **Strategy** позбавляє від жахливих умовних конструкцій.
- **Facade** ховає складність системи за простим інтерфейсом.
